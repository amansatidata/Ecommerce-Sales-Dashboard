# Contributing to E-Commerce Sales Performance Dashboard

Thanks for your interest in this project! This is primarily a personal portfolio project, but suggestions, corrections, and improvements are welcome.

## How to Contribute

1. **Fork** this repository
2. **Create a branch** for your change
   ```
   git checkout -b fix/short-description
   ```
3. **Make your changes**
   - Dashboard updates: edit `dashboard.pbix` in Power BI Desktop
   - Data updates: place new/cleaned files in the `data/` folder
   - Documentation: edit `README.md` or files in `docs/`
4. **Commit your changes** with a clear message
   ```
   git commit -m "Fix: corrected profit margin calculation in DAX measure"
   ```
5. **Push and open a Pull Request** describing what you changed and why

## Types of Contributions Welcome

- Bug fixes (e.g., incorrect DAX measures, broken visuals, data errors)
- Documentation improvements (README clarity, added context, typo fixes)
- New visualizations or dashboard enhancements
- Data cleaning improvements or additional data sources
- Performance optimizations (query folding, model size reduction, etc.)

## Guidelines

- Keep commit messages clear and descriptive
- If modifying the `.pbix` file, briefly describe what changed in your PR (Power BI files are binary, so diffs aren't readable)
- If adding new data, make sure it doesn't contain any sensitive or personally identifiable information
- Test that visuals/filters still work correctly before submitting a PR
- Be respectful and constructive in discussions

## Reporting Issues

If you spot a bug, incorrect insight, or have a suggestion, please open an [Issue](../../issues) with:
- A clear description of the problem or suggestion
- Steps to reproduce (if applicable)
- Screenshots, if relevant
