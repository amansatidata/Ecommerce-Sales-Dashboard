# Security Policy

## About This Project

This is a personal data analytics portfolio project (Power BI dashboard) built using a public/sample e-commerce dataset. It does not collect user data, run backend services, or process live/production data.

## Supported Versions

This is a single-version portfolio project. Only the latest version on the `main` branch is maintained.

| Version | Supported |
|---------|-----------|
| main    | ✅ |

## Data & Privacy

- The dataset used in this project is a publicly available / sample e-commerce dataset and does **not** contain real customer personally identifiable information (PII).
- No credentials, API keys, connection strings, or secrets are included in this repository.
- If you fork or reuse this project with your own data, **do not** commit sensitive data, credentials, or `.pbix` files containing embedded live database connections.

## Reporting a Vulnerability or Issue

If you find something concerning in this repository — such as:
- Accidentally exposed credentials or connection strings
- Sensitive data that shouldn't be public
- A broken/malicious link or file

please report it by:
1. Opening a private [Security Advisory](../../security/advisories/new) on GitHub (preferred), or
2. Contacting me directly via [LinkedIn](https://linkedin.com/in/aman-sati) or email at amansati918@gmail.com

Please avoid opening a public Issue for sensitive findings.

## Response

As this is a personal portfolio project, response times are best-effort, but reports will be reviewed and addressed as quickly as possible.
